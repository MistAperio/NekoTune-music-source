## 📖 简介

本仓库致力于收集、整理和分享 **LX Music（洛雪音乐）**、**CeruMusic（澜音）**、**IKUN Music** 等第三方音乐播放器的音源资源。

所有音源均来源于网络公开渠道，仅供 **个人学习与技术交流** 使用，请尊重版权，支持正版音乐。

## 🚀 快速开始

### LX Music（洛雪音乐）

1. 打开软件 → **设置** → **音乐来源** → **自定义源管理**
2. 点击 **导入**，输入音源链接或选择本地 `.js` 文件
3. 选择对应音源即可开始播放

### CeruMusic（澜音）

1. 打开软件 → **设置** → **插件管理**
2. 点击 **导入插件**，选择 `.js` 文件
3. 启用插件，选择对应音源即可

### IKUN Music

1. 打开软件 → **设置** → **音源管理**
2. 点击 **导入音源**，输入链接或选择文件
3. 刷新列表即可使用
